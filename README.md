# ⚛️ Personal Portfolio — React + Vite + Tailwind

A rebuild of my personal portfolio using modern web technologies, maintaining the same visual identity with improved code structure.

## 🚀 Live Demo
[View Portfolio](https://marcooab.github.io/personal-portfolio/)

## 🛠️ Tech Stack
![My Skills](https://skillicons.dev/icons?i=html,css,js,react,tailwind,vite)

## ✨ Features
- Built with React components
- Tailwind CSS v4
- Typed.js text animation with dynamic icons
- Custom hooks (useScrollPosition, useMobileMenu)
- Projects carousel with animated cards
- Skills section with tech icons
- EmailJS contact form with toast notification
- Responsive design
- Deployed on GitHub Pages
